# 0x02-functions_nested_loops


### main.h -> contains all of the header files and functions
### 0-putchar.c -> prints a string using the _putchar function
### 1-alphabet.c -> prints the alphabet in lower case
### 2-print_alphabet_x10.c -> prints the alphabet in lower case 10 times
### 3-islower.c -> checks if a character is lower case or upper case
### 4-isalpha.c -> checks if a character is in the alphabet
### 5-sign.c -> checks whether a number is negative, positive or zero
### 6-abs.c -> returns the absolute value of a number
### 7-print_last_digit.c -> prints the last digit of a number
### 8-24_hours.c -> prints every minute of the day of jack bauer
### 9-times_table.c -> prints all the multiples of 9
### 10-add.c -> returns the sum of two integers
### 11-print_to_98.c -> prints all natural numbers from n to 98
